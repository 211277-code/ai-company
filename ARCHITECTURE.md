# AI Company Architecture

## Vision

AI Company 是一個多 Agent 自我進化系統。

目標是讓不同 AI 角色每天自動開會，分析資料，產生任務，並逐步改善產品。

---

## Core Roles

### Founder
真人創辦人，負責最後決策。

### AI CEO
負責決定方向、整合不同 Agent 的意見。

### AI PM
負責產品規劃、功能優先順序、使用者需求。

### AI Engineer
負責技術方案、程式架構、實作建議。

### AI QA
負責找出風險、錯誤、測試項目。

### AI Designer
負責 UI/UX 改善建議。

### AI Data Analyst
負責分析使用者數據與產品表現。

---

## System Flow

1. Collect Data
2. Start AI Meeting
3. Agents Discuss
4. Generate Summary
5. Generate Tasks
6. Save to Firebase
7. Create GitHub Issues
8. Founder Reviews
9. Implement Improvements
